# Splitter

A mobile app for tracking and splitting group expenses. Perfect for trips, shared housing, or any situation where multiple people need to track shared costs.

## Documentation
Full documentation is available in the `docs/` directory:

- [**Features**](docs/features.md): Details on Expenses, Settlements, and Offline Mode.
- [**Architecture**](docs/architecture.md): Software stack and project structure.
- [**Setup**](docs/setup.md): Development environment setup.
- [**Database**](docs/database.md): Firestore schema reference.
- [**Backend**](docs/backend.md): Cloud Functions and Security.

## Quick Start
1.  **Clone**: `git clone ...`
2.  **Setup**: Ensure you have Flutter installed and `google-services.json` (Android) from Firebase.
3.  **Run**: `flutter run` (Android is the primary test platform).
